openvpn job

This job is installed to /var/vcap/jobs/openvpn. It depends on openvpn.

Properties

The following properties may be configured for this job…

openvpn.port (integer)

Bind Port for the server

Default: 1194

openvpn.extra_config (text)

Custom OpenVPN configuration statements

openvpn.local (string)

Bind IP for the server

Default: "0.0.0.0"

openvpn.server (string)

VPN IP and netmask

openvpn.routes (string[])

Routes for the local routing table

openvpn.push_routes (string[])

Routes to push to connecting clients

openvpn.cipher (string)

Cipher for encrypting packets

Default: "BF-CBC"

openvpn.keysize (integer)

Size of cipher key in bits

Default: 256

openvpn.ca_crt (text)

CA Certificate

openvpn.server_crt (text)

Server Certificate

openvpn.server_key (text)

Server Key

openvpn.dh_pem (text)

Diffie-Hellmann Key

openvpn.crl_pem (text)

Certificate Revocation List

openvpn.ccd (array[])

A list of Client Configuration Directives

openvpn.iptables (string[])

IPTable rules to manage

Example: ["POSTROUTING -t nat -s 192.0.2.0\/24 -d 10.10.1.0\/24 -j MASQUERADE -m comment --comment 'vpn -> private lan'","POSTROUTING -t nat -s 192.0.2.0\/24 -d 10.10.2.100\/32 -j MASQUERADE -m comment --comment 'vpn -> internal backup server'"]

openvpn.client_config (array[])

A list of Client Configuration Connections
